# custom_shell

A small custom shell written in Rust.

## Features

- Execute external commands
- Builtin commands: `cd`, `pwd`, `exit`
- Basic relative/absolute path handling

## Requirements

- Rust toolchain (stable)

## Build

```bash
cargo build --release
```

## Run

```bash
cargo run --release
# or
./target/release/custom_shell
```
